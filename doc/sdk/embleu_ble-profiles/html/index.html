Getting Started

em | bleu software stacks provides connectivity solutions for Bluetooth. This site contains technical information for em | bleu software stack products. Documents are organized into the following types of documents:

  • Developer’s Guide (DG): information about architecture and behavior, compilation, and configuration

  • User’s Guide (UG): application operational usage description

  • API Documentation: functional description of Application Programming Interface
