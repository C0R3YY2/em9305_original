◆ PalRadioInitTxPower()

void PalRadioInitTxPower ( void  )

Initialize RF transmit output min and max power.

This function initializes internal min and max values for the transmit output power. By default, the min power is set to -57dBm while the max power depend on both the available power supply and the value written into the locked power register.

)

Set next Rx packet offset.

Parameters
offsUsecOffset in microseconds.
rxTimeoutUsecRx correlation timeout.
Returns
true if successful. false if failed.
