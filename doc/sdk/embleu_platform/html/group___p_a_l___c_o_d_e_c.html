Configure data path.

Parameters
dirDirection.
vsCfgLenLength of pVsCfg buffer.
pVsCfgVendor specific configuration buffer.
Returns
true if valid, false otherwise.

This function is called by the LL when the host invokes HCI_Configure_Data_Path with a Data_Path_ID=1 (LL_ISO_DATA_PATH_VS).
