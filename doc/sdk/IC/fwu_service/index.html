Firmware Update Service

The Firmware Update Service library is an implementation of the EM Firmware Update service .

API

group fwu_svc

Firmware Update GATT based service for EM Bleu Bluetooth Stack.

Defines

FWUSVC_START_HDL 0x200

Firmware Update Proprietary Service Start Handle.

UUID16_FWU_SERVICE 0xA888u

16-bit UUID for Firmware Update service

UUID16_FWU_CMD 0xA9A0u

16-bit UUID for Firmware Update Command characteristic

UUID16_FWU_DATA 0xAAB8u

16-bit UUID for Firmware Update Data characteristic

UUID_EM_BASE_PART1 0x28u, 0xD4u, 0x1Bu, 0xE0u, 0x7Au, 0xFAu, 0xC0u, 0xADu, 0xE8u, 0x11u, 0x4Du, 0x45u

Base UUID: 81cfXXXX-454D-11e8-adc0-fa7ae01bd428.

Base UUID PART1”: 454D-11e8-adc0-fa7ae01bd428

UUID_EM_BASE_PART2 0xCFu, 0x81u

Base UUID PART2”: 81cf.

UUID_EM_BUILD(uuid16) UUID_EM_BASE_PART1, UINT16_TO_BYTES(uuid16), UUID_EM_BASE_PART2

Macro for building UUIDs.

Proprietary services.

Proprietary Command characteristics.

Proprietary Data characteristics.
