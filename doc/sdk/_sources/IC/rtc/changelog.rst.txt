Change Log
------------

Version 1.1.0
^^^^^^^^^^^^^

* Add gRTC_Config external declaration in public interface (in file `rtc.h`).

Version 1.0.0
^^^^^^^^^^^^^

* Initial Version.
* Support Alarm and Short Alarm.
* RTC Event with callback (Ready, Alarm, Short Alarm, End of Day).
* RTC can be used with LF-RC, LF-XTAL with or without calibration.
* Two format of time supported, elapsed time from reference time (1st January, 2000) or date & time.
* RTC compute the day of the number.
* First day of the week can be configured (Monday or Sunday).
