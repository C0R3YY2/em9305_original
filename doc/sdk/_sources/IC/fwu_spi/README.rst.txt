Overview
--------

This library implements the Firmware Updated functionalities for SPI transport.
It defines the command parser used in the `firmware_updater <../firmware_updater/index.html>`_ as well as `fwu_target <../fwu_target/index.html>`_ applications.

Refer to the `Firmware Update <../../sections/FirmwareUpdate.html>`_ section of the SDK Documentation for additional information on the firmware update functionality.
