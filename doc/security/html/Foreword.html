ForewordÔÉÅ

This documentation is a preliminary draft version of EM7198 crypto library documentation. It is a direct copy of EM9305 crypto lib documentation.

Some content is known to be out of date:

  • The section on crypto containers needs to be updated to reflect the new containers structure on EM7198 (128 containers of 4*128 bits, instead of 256 containers of 2*128 bits)

  • Due to the ECC(Error Correcting Code), the AES driver will be updated. It implies that more error status will be possible on EM7198.

  • In order to comply with Misra and the code metrics requirements, all the libs will be reworked and recompiled. It implies that the performances will be slightly different on EM7198. The given timings are just indicative.

In general, the performances are approximative. They can vary depending on the cache, the chip frequency. For AES modes, the performances of individual functions usually depend on the previous calls and how many bytes were bufferized. The given time are thus indicative.
