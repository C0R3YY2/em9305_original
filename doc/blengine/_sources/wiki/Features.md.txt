# Features

The BLEngine comes along with the following features and capabilities:

- Modular Architecture
- Multi-Threaded Processing
- Configuration based
- Generic Fast Registry for stream parsing
- Serial transport and virtual transport
- Built-in CLI interface
- Generic engine for easy-to-use communication with any device
- Full HCI Protocol implementation (following the Bluetooth Core Specification v5.2 and later versions)
- Bluetooth GAP and GATT Layer implementation (work in progress)
